# 腾讯云 · 联络中心含 UI 解决方案
  TCCCCallKit 是腾讯云联络中心推出的一款电话外呼、接听的云联络中心UI组件，通过集成该组件，你只需要编写几行代码就可以为你的App添加外呼手机、电话等音频通话等功能。目前 TCCCCallKit 支持 Android 、iOS 等多个开发平台。
